# cairn-sandbox

Throwaway repository for testing [Cairn](https://github.com/dotnetemmanuel/Cairn).

**Safe to break.** Anything here exists only to exercise Cairn's dashboard,
review, and stacked-PR workflows against a real GitHub remote. Branches, PRs,
comments, and history may be created and destroyed at will.

## Layout

- `src/greeting.py` — sample Python module (now with friendlier, time-aware greetings)
- `src/math_utils.js` — sample JavaScript module
- `src/inventory.py` — sample inventory module
- `.github/workflows/ci.yml` — trivial CI so PRs get real check statuses
